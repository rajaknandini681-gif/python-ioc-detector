# Python IOC Detection Tool

A beginner SOC analyst project built with Python for detecting Indicators of Compromise (IOCs) from log files.

## Features

- IOC Detection
- Failed Login Detection
- Encoded PowerShell Detection
- Log Analysis
- Alert Reporting

## Technologies Used

- Python
- Regex
- Threat Hunting Basics

## How To Run

```bash
python detector.py
```

## Sample Detections

- Malicious IP Addresses
- Malware Hashes
- Suspicious PowerShell Commands
- Failed Login Attempts
